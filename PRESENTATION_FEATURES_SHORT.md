# LIMS - Feature Summary (Presentation Version)

## 🎯 Main Modules

### 1. **Job Management**
- Create and track calibration jobs
- Multiple statuses and view modes
- Comprehensive job details

### 2. **Job Assignment System**
- One-click job assignment to staff
- Auto status change & start date
- Assignment notifications
- Role-based permissions

### 3. **Service Request System**
- Public service request form
- Auto-complete for customers/equipment
- Convert requests to jobs

### 4. **Customer Management**
- Customer database
- Search and filtering
- Multiple view modes

### 5. **Equipment & Master Lists**
- Equipment database
- Master lists (Equipment, Manufacturer, Model, Calibration Methods)
- Excel synchronization

### 6. **PDF Reports**
- Customizable PDF templates
- Dynamic components
- Formula calculations

### 7. **Document Management**
- Form builder
- Document templates
- Workflow management

### 8. **User & Security**
- Role-based access control
- User management
- Permission system

### 9. **Notifications**
- Real-time notifications
- Job assignment alerts
- Notification center

### 10. **Data Export**
- Excel export
- Excel sync for master lists

### 11. **Settings**
- Job ID configuration
- PDF settings
- System configuration

---

## 🔄 Key Workflows

**Service Request → Job → Completion**
1. Customer submits request
2. Admin converts to job
3. Assign to staff
4. Track progress
5. Generate reports

---

## 💻 Technology Stack

- **Frontend**: React + TypeScript
- **Backend**: Firebase (Firestore, Auth, Storage)
- **UI**: Tailwind CSS
- **Real-time**: Firestore listeners
- **PDF**: Puppeteer server
