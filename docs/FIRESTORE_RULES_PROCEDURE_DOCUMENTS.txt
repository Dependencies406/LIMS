# Quick Firestore Rules for Procedure Documents

Copy and paste this into your Firestore Rules in Firebase Console:

```javascript
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    
    // Procedure Documents Collection
    match /procedureDocuments/{documentId} {
      // Allow read to all authenticated users
      allow read: if request.auth != null;
      
      // Allow write to all authenticated users (restrict to admin in production)
      allow write: if request.auth != null;
    }
  }
}
```

## Instructions:
1. Open Firebase Console
2. Go to Firestore Database → Rules tab
3. Add the rules above to your existing rules
4. Click "Publish"
5. Wait a few seconds for rules to propagate

## For Production:
Replace the `allow write` line with:
```javascript
allow write: if request.auth != null && 
               get(/databases/$(database)/documents/users/$(request.auth.uid)).data.role == 'admin';
```
