🔍 How It Works

PDF Generation Flow:

  1. User clicks PDF button
  2. Service creates HTML with job data and styling
  3. HTML added to DOM (hidden, off-screen)
  4. html2canvas converts HTML to canvas image
  5. jsPDF converts canvas to PDF document
  6. PDF downloaded or previewed
  7. Cleanup - temporary HTML removed

Time: Usually 1-3 seconds depending on content size

You can customize PDFs by modifying src/services/pdfService.ts:

  • Fonts: Change font family in the style section

🐛 Troubleshooting

If Thai characters don't display:

  1. Check that Google Fonts is loading (inspect network tab)
  2. Clear browser cache
  3. Verify internet connection (fonts load from Google)

If PDF quality is low:

Increase the scale in pdfService.ts:

const canvas = await html2canvas(container, {
  scale: 3,  // Change from 2 to 3 for higher quality
  useCORS: true,
  logging: false,
  backgroundColor: '#ffffff'
});

If PDF is too large:

Decrease the scale or optimize images in the content.
