🔧 Technical Specifications

Technology Stack

  • Frontend: React 18 + TypeScript + Vite
  • Styling: Tailwind CSS
  • Backend: Firebase (Auth, Firestore, Storage)
  • PDF Generation: Node.js + Puppeteer + Express
  • State Management: React Context API + Custom Hooks

Data Flow

Frontend (React) ↓ Service Layer (TypeScript) ↓ Firebase (Firestore/Auth/Storage) PDF Generation: Frontend → pdfService.ts → PDF Server (port 3001) → Puppeteer → PDF File

Storage Strategy

  • User Accounts: Firebase Authentication
  • User Profiles: Firestore /users collection
  • Jobs Data: Firestore /jobs collection
  • Customer Data: Firestore /customers collection
  • Company Info: Firestore /settings/companyInfo document
  • PDF Settings: Browser localStorage (per-user)
  • Company Logo: Firebase Storage /company/logo/
